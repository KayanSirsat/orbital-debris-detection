# Orbital Debris & Satellite Detection

An end-to-end computer vision pipeline for space situational awareness.
This project demonstrates synthetic data generation, object detection
using YOLOv8, and geometric pose estimation with OpenCV.

## Tech Stack
- Python
- Ultralytics YOLOv8
- OpenCV
- NumPy

## Pipeline
1. Synthetic orbital image generation
2. YOLO-based detection
3. Sensor noise simulation
4. Orientation estimation via geometry
